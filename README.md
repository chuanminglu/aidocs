# AI文档管理系统

这是一个支持双维度索引（时间+专业）、强大模板管理、基础分类/标签/搜索功能的文档管理系统。

## 特性

- 📁 双维度索引：时间+专业领域
- 📄 模板管理系统：可扩展、可视化、支持大纲导航
- 🏷️ 分类和标签功能
- 🔍 基础搜索功能
- 📊 三面板界面：目录树+大纲导航+内容区+预览区

## 快速开始

### 环境要求
- Node.js 14+
- 现代浏览器

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm start
```

访问 http://localhost:8080 查看系统
